Install
------------------------------------------------------------------------------

``jenkins_playground`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install jenkins_playground

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade jenkins_playground
